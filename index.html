How This Dynamic Blog Works

System Info

Super Easy Blog Management:

  1. Use the "Editor" tab to create your post
  2. Save the HTML file to your blog folder
  3. Add the filename to the allFiles array in index.html (one simple line!)
  4. Commit and push to GitHub Pages

The system automatically knows which files are blog posts vs. static pages!

Current files being tracked: about.html, contact.html are automatically excluded. Everything else becomes a blog post!

About This Blog

Welcome to my dynamic GitHub Pages blog! This blog automatically loads and displays posts from individual HTML files.

Features:

  • Automatic post loading and excerpt generation
  • Built-in post editor
  • Clean black and yellow design
  • Fully hosted on GitHub Pages
  • No database required

The system works by maintaining a list of post filenames and dynamically loading content from each file to create excerpts on the home page.
