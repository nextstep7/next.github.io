About This Enhanced Blog

Welcome to my enhanced dynamic GitHub Pages blog! This blog automatically loads and displays posts from individual HTML files.

New Features:

  • Optional post images with automatic resizing
  • External reference links for sources and related content
  • Enhanced post editor with improved layout
  • Responsive design for mobile and desktop

Existing Features:

  • Automatic post loading and excerpt generation
  • Built-in post editor
  • Clean black and yellow design
  • Fully hosted on GitHub Pages
  • No database required

The system works by maintaining a list of post filenames and dynamically loading content from each file to create excerpts on the home page.
